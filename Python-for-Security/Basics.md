## Python   
Dynamically typed ,general purpose programming language. Supports an object orientedd programming approach , it is very easy to implement and understand.It is open source language, it has a very big library as well as modules ,possible to integrate with other languages also within python, it is vastle used in many things.  

**Modules**  &rarr; is like a code library which can be used to borrow the code written by someone else.  
`#` &rarr; By using this the sentence/line becomes comment which is not executed by the compiler/interpreter.  
`\n` &rarr; Escape sequence/ Character for new line.  
`\"` &rarr; Escape sequence.  

**DATA TYPES** 
Specify the type of value a variable holds.required  in programming to do operations without causing an error.
Default data-types  
- Numeric data --> int, float, complex.
- Text data --> Str(string).
- Boolean data ---> True/False.
- Sequenced data --> List ,tuple.
- Mapped data --> Dict(dictionary).
